## Overview
Bug Bounty Radar (bbradar.io) is a fast, comprehensive aggregator for publicly listed bug bounty and vulnerability disclosure programs. Updated multiple times per hour with powerful filtering, target monitoring, platform intelligence, and Pro API access, it helps you discover programs that match your skills and tech stack across both Web2 and Web3 ecosystems.

## Key Features

- **Unified Feed** - Track public programs across major Web2, Web3, contest, audit, and VDP platforms in one place
- **Latest Targets** - Monitor scope additions, removals, target changes, and full target lists across supported platforms
- **Live Notifications** - Get Discord alerts for new programs and target changes as they happen
- **Platform Intelligence** - Compare platform momentum, popularity, reward profiles, VDP share, daily snapshots, and 90-day trends
- **Community Voting** - Sign in with Google to vote on programs and platforms, with public popularity labels and rankings
- **Report Count Awareness** - Surface report-count metadata where available, with Pro access for premium counts
- **Pro API** - Generate an API key for programmatic access to programs, targets, target changes, and report-count metadata
- **Language Filters** - Filter programs by Python, JavaScript, TypeScript, Go, Rust, Solidity, Cairo, Clarity, Move, C, C++, Objective-C, Kotlin, Java, Sway, Swift, Ruby, and more
- **Powerful Discovery** - Filter by type, tag, platform, language, target, asset identifier, report count, votes, and VDP visibility
- **Advanced Tag System** - Filter by AI/ML, blockchain-dlt, CIDR, domain, executable, hardware, IP address, mobile, package, smart contract, source code, wildcard, and many more
- **Bounty Awareness** - Surface bounty amounts and reward ranges when available
- **Pro Subscription** - Unlock live alerts, Discord delivery, full target visibility, report counts, and API access
- **Dark/Light Mode** - Choose your preferred theme for comfortable browsing
- **Clean UX** - Minimal, fast interface built for efficient scanning and research
- **Web2 and Web3 Views** - Switch focus between ecosystems with a single click

## Pro

Pro adds live monitoring and programmatic access on top of the free public feed:

- **Live program notifications** - Get alerted when a new public program goes live
- **Live target-change notifications** - Track scope additions, updates, removals, and eligibility changes
- **Full target visibility** - Search complete target lists for every supported program
- **Report counts** - See report-count metadata when platforms expose it
- **Discord access** - Connect Discord for Pro roles and live notification channels
- **API keys** - Manage keys from your account and call `/v1/pro/programs`, `/v1/pro/programs/{id}/targets`, and `/v1/pro/targets/changes`

## Platform Intelligence

The Platforms page is now a live comparison dashboard, not just a list of sources:

- Platform popularity rankings from community votes
- Daily platform snapshots and 90-day new-program trendlines
- Active program counts, recent launches, VDP share, and reward profile summaries
- Platform-specific pages for quick filtering into the main program feed
- Free public access, with sign-in only needed when submitting votes

## Why bbradar.io
Manually checking multiple platforms is slow and inefficient. bbradar.io gives you a single, unified, real-time view of new public bug bounty and VDP programs across major platforms. Discover targets faster, monitor scope changes, compare platform activity, and spend your time on research instead of tab hopping.
